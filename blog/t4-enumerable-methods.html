Saturday: The Enumerables: group_by

March 21, 2015

The enumerable "group_by" breaks a collection into groups based on some given criteria. It returns these groups as a hash where each key represents each group and the value is an array containing the member of each group.

Example Ruby Code: (0..15).group_by {|number| number % 3}

Output: {0=>[0, 3, 6, 9, 12, 15], 1=>[1, 4, 7, 10, 13], 2=>[2, 5, 8, 11, 14]}

The numbers 0-15 are passed through the criteria (number % 3). The output is a hash where the keys are the 3 groups (remainder of 0, 1 or 2) and the values are the associated members of the group which are the numbers 0-15 that return 0, 1 or 2.
