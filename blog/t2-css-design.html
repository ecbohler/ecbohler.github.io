Sunday: Inline Block Display vs. Inline Display

March 8, 2015

So what does it mean to display an element using inline-block versus inline?

A Block Display --> This element will be displayed across the full width of the page no matter the amount of content. Vertically stacked on top of one another.

An Inline-Block Display --> This element will be displayed next to each other on one line. Horizontally stacked blocks.

An Inline Display --> This element will be displayed next to each other but not as blocks because they do not keep their dimensions. Horizontally stacked blocks.
