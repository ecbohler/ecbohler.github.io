AWS and Wordpress Permissions

September 21, 2015

Lately, I've been learning how to use Wordpress by building websites and web applications. As we all know, when learning a new framework, there are plenty of minor nuances that prevent us from moving forward. Since google is our friend, we can find most of the answers to these "simple" questions.

My latest hiccup involved folder permssions. By default, AWS grants permission to the ec2-user in the AMI. When attempting to perform upgrades or updates to plugins via Wordpress, I was prompted to enter my server information to perform a transfer of the updates via FTP. Well AWS doesn't connect in this manner so I was stuck.

However, after some surfing, searching and reading, I found the answer to be quite simple; change the permissions of that folder to allow Wordpress to access it.

sudo chown -R apache:apache /var/www/html
