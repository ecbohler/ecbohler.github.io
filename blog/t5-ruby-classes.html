Class Fruit_salad {

def initialize

@apples = apples

@peaches = peaches

@berries = berries

}

end

def cut

Print “Cut all of the #{@apples}, #{@peaches} and #{@berries} into eigths”

end

def mix {

glass = string.gets.chomp

if glass == “Icy”

print “#{@apples}, #{@peaches} and #{@berries} can be added to the ice.”

else

print “Don’t Mix This Yet”

end

end

}

The instance variables for this example are @apples, @peaches and @berries. Instance Variables allow you to assign a variable and have access to that variable throughout the class.


The instance methods for this example are: def initialize, def cut and def mix. In stance methods are groups of behaviors that perform an action on an object
